### 1.2 Component Breakdown

**Title Bar (40px height)**
- Logo + App Name (left)
- Window controls: Theme toggle, Settings, User menu (right)
- Custom chrome for cross-platform consistency

**Navigation Rail (56px width)**
- Icon-only by default
- Hover reveals tooltip with label
- Active item: orange left border + orange icon
- Bottom section: Settings + Profile (always visible)

**Tab Bar (36px height)**
- Project selector dropdown (left)
- Document tabs with close buttons
- Overflow menu for many tabs
- New tab button (+)

**Editor Area (flexible)**
- Markdown rendering with syntax highlighting
- Inline style indicators (squiggly underlines)
- Line numbers (optional)
- Minimap (optional, wide screens only)

**Status Bar (28px height)**
- Word count
- Estimated reading time
- Style compliance score
- AI connection status
